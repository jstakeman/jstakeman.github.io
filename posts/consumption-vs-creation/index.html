Consumption vs Creation

Consumption is very easy. We start as infants. Capitalism wouldn't be quite so appealing if it were hard to get people to buy things.

You would think, that in order for there to be some balance, the creation of things would be a comparable process. How else would we find some much to consume if it wasn't also being created?

Is this all just a side effect of the Industrial Age? Did we use to be a world of creators and consumers in equal measure? Did the steam engine tilt the scales so far in the direction of creation that it perversely made it harder to create things?

I sometimes find myself staring at media that I have no interest in. Last night I spent an hour listening to Friends with my eyes closed. Surely a few minutes of typing would be easier than that particular endurance test. Listening to the holiday armadillo for the umpteenth time can not be an easier task than writing this.
